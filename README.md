# Telegram Helpdesk Bot

Simple telegram bot that responds to help requests in a group chat and continues the conversation in private messages.

## Fetures

- Detects help-related keywords in groupchat
- Sends instructions for common issues (printer, internet, sound, projector
- Encourages users to restart devices before escalating to a human operator
- Option to connect to a human operator

## What to expect in the future

- Respons only to issues related to devices stored in a data base
- Store all reported problems in a database for analysis
- Support for more complex troubleshooting steps
- Optional AI integration for smarter responses 

## Requirements
- Python 3
- python-telegram-bot
